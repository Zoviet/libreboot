# Свободные дистрибутивы GNU/Linux и Libreboot

Libreboot позволяет избавиться от несвободного ПО на самом низком уровне. Логично, что и все остальное ПО при использовании Libreboot должно быть полностью свободным, иначе смысл всех действий несколько теряется. По этой причине создатели Libreboot настоятельно рекомендуют использовать на компьютерах с Libreboot исключительно свободные дистрибутивы.

[Список свободных дистрибутивов GNU/Linux поддерживается Free Software Foundation](https://www.gnu.org/distros/free-distros.html) исходя из соблюдения идеи свободного ПО: 
>"Слова “свободная программа” означают программу, которая уважает свободу и сообщество пользователей. В двух словах это значит, что у пользователей есть свобода выполнять, копировать, распространять, изменять и улучшать программу. Таким образом, понятие “свободная программа” относится к свободе, а не к стоимости. Чтобы верно воспринимать эту концепцию, вы должны понимать слово “свободный” в том смысле, который заключается в выражении “вольная речь”, а не “бесплатное пиво”." См. [https://www.gnu.org/philosophy/free-sw.html](https://www.gnu.org/philosophy/free-sw.html)

В отношении дистрибутивов соблюдение принципов свободного ПО означает, что дистрибутив не может и не должен содержать в себе никаких несвободных компонентов, что несколько шире, чем требование того, чтобы весь код был открытым. Свободные дистрибутивы - это дистрибутивы не только с полностью открытым, но и с полностью свободным кодом. 

FSF в комментариях к списку подчеркивает, что "в мире свободы работает не вся аппаратура; на сайте каждого из дистрибутивов должно указываться, какая аппаратура поддерживается". Этот нюанс в приложении к системам на основе Libreboot требует разбора каждого из существующих свободных дистрибутивов. 

## Dragora GNU/Linux-Libre

* Сайт проекта:[https://dragora.org](https://dragora.org)
* Репозиторий проекта: [https://notabug.org/dragora](https://notabug.org/dragora)
* Группа проекта: [https://gnusocial.no/group/dragora](https://gnusocial.no/group/dragora)
* Текущий TODO-list: [http://git.savannah.nongnu.org/cgit/dragora.git/tree/TODO](http://git.savannah.nongnu.org/cgit/dragora.git/tree/TODO)
* Freenode: #dragora

Находится в стадии Beta, текущая версия ядра GNU Linux libre 4.19.78. Особенность дистрибутива, позиционируемого как "простой и элегантный", в использовании далеко не самых распространенных менеджеров пакетов. Есть 32-битная версия. Для большинства конфигураций "из коробки" работать не будет, документация проекта для неподготовленных пользователей находится на зачаточном уровне. Использовать для ежедневного использования тяжело, 32-битная Live система на Libreboot x60s сразу не запустилась. Подходит скорее для энтузиастов.

## Dyne:bolic

* Сайт проекта:[https://www.dyne.org/software/dynebolic/](https://www.dyne.org/software/dynebolic/)
* irc.dyne.org: #dyne

Дистрибутив, позиционируемый как свободная мультимедийная студия "в коробке", т.е. в виде загрузочного образа. Для постоянной установки малопригодна.
